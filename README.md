# Translator
Практика "Транслятор" (Курс Nand2Tetris) - 6. Ассемблер 

## Условие задачи:

Продолжайте в том же проекте со стартовым кодом.

Реализуйте основную логику транслятора из символьного ассемблера в машинный код платформы Hack.

Действуйте небольшими шагами — начните с трансляции отдельных инструкций. В файле HackTranslatorTests есть отдельные тесты на трансляцию C-инструкций, А-инструкций и всей программы целиком. С их помощью вы сможете проверять корректность каждого своего шага.

В реальной ситуации, вам пришлось бы создавать подобные тесты самостоятельно. Но точно так же было бы удобно разделить работу на отдельные шаги, и создавать тесты на очередной шаг перед тем, как переходить к следующему. Это упрощает отладку и позволяет повысить уверенность в написанном коде.

Значение битов A-инструкции:
15	14..0
0	address

Значение битов C-инструкции:
15..13	12	11..6	5..3	2..0
111	a	cccccc	ddd	jjj
